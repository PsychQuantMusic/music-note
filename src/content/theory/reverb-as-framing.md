Reverb 的傳統定義是「空間模擬」。但這個定義太淺了。

**Reverb 不是再現空間，是建構認知框架。**

---

## 兩種思考方式

### 傳統：Reverb = 空間再現

```
問題：這個聲音在什麼空間發出？
答案：大廳 → 用 Hall reverb
      小房間 → 用 Room reverb
      浴室 → 用 Tile reverb
```

這是物理模擬的思路。

### 新觀點：Reverb = Framing

```
問題：我要聽眾覺得這個聲音屬於什麼世界？
答案：史詩感 → 長 decay
      親密感 → 短 decay 或 dry
      夢幻感 → Shimmer
      復古感 → Plate
```

這是認知建構的思路。

---

## 證據：不存在的空間

有些 reverb 不對應任何真實空間：

| Reverb 類型 | 真實空間？ | 但它建構的「世界」|
|------------|----------|-----------------|
| Shimmer | 不存在 | 天堂、夢境、超自然 |
| Reverse reverb | 物理上不可能 | 時間倒流、詭異 |
| Gated reverb | 人造的 | 80 年代、Power |
| Infinite reverb | 不存在 | 永恆、消融、Ambient |

**這些 reverb 不「模擬」任何東西，但它們創造意義。**

## 外部資料佐證

### Dan Worrall（FabFilter Pro-R 2 Introduction）

Dan Worrall 在介紹 Pro-R 2 時的開場白，幾乎是這個理論的完美印證：

> 「這是一個很有喚起力的聲音⋯⋯但**它喚起的記憶，很大程度取決於它所在的聲學空間**：如果我調出一個小房間⋯⋯把聲源拉近⋯⋯這個聲音可能讓你想起**童年在客廳學鋼琴**⋯⋯但如果用更大的空間⋯⋯更亮的響應⋯⋯它可能開始喚起**在學校禮堂唱詩歌的記憶**⋯⋯這有**完全不同的情感衝擊**。」

注意他說的是 **memories**（記憶）和 **emotional impact**（情感衝擊），而不是「這模擬了多大的空間」。

### Dan Worrall（Beginner's Guide to Reverb）

關於沒有 reverb 的聲音：

> 「完全沒有 reverb 的信號會顯得相當不自然，**彷彿漂浮在虛空中，沒有脈絡來賦予它們尺度，沒有深度或維度感**。」

關於 reverb 作為敘事工具：

> 「**改變 reverb 告訴聽眾我們已經移動到不同的空間，就像走過一扇門進入另一個房間**⋯⋯在歌曲的不同部分切換到不同的 reverb 類型，有助於**增加情緒的動態範圍，在段落之間創造更大的對比**。」

這直接支持「Reverb 的變化 = 敘事的節奏」這個觀點。

### Chuck Sutton（iZotope 9 Reverb Mistakes）

一個精彩的類比：

> 「**把 reverb 想像成斜體字。** 現在想像讀一本書，每個字都是斜體。沒有任何東西會被真正強調，因為一切都是同樣的興奮程度。」

這是**語言修辭**的概念——reverb 是聲音的修辭手段，不是物理模擬。

### In The Mix（Are You Using the Wrong Reverb）

關於 reverb 的多重用途：

> 「你需要問自己：**我是想增強氛圍，還是想做一些瘋狂的聲音設計？**」

這暗示 reverb 不只有「模擬空間」一種用法，它可以是意義建構的工具。

## Reverb 參數的 Framing 意義

### Decay Time（殘響長度）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| < 1s | 小空間 | 親密、現代、直接 |
| 1-2s | 中型空間 | 自然、平衡 |
| 2-4s | 大空間 | 史詩、莊嚴、正式 |
| > 4s | 巨大空間 | 超自然、神聖、失真現實 |

### Pre-delay（預延遲）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 0-20ms | 聲源在空間中央 | 聲音「在」空間裡 |
| 20-50ms | 聲源靠近聽者 | 聲音「前面」，空間「後面」|
| > 50ms | 明顯分離 | 人造感、效果器感 |

Pre-delay 分離 dry 和 wet，讓聲音「站在」空間前面。

### High Frequency Damping（高頻衰減）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 亮（少衰減）| 反射面光滑 | 現代、數位、冷 |
| 暗（多衰減）| 空氣吸收 | 復古、溫暖、自然 |

### Wet/Dry Mix

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 10-20% | 輕微空間感 | 真實、自然 |
| 30-50% | 明顯空間感 | 有距離、有氛圍 |
| > 50% | 空間主導 | 夢幻、模糊、非現實 |
| 100% wet | 只有反射 | 完全脫離現實 |

## Reverb 類型的文化符號

不同 reverb 類型攜帶歷史/文化意義：

### Plate Reverb

```
歷史：1950s-80s 錄音室標配
聲音：密集、亮、金屬感
Framing：復古、經典、「錄音室」感
使用：想讓聲音聽起來「像唱片」
```

### Spring Reverb

```
歷史：吉他音箱內建
聲音：彈簧共振、有「波動」
Framing：復古搖滾、Surf、Lo-fi
使用：想讓聲音有「類比」質感
```

### Hall Reverb

```
歷史：模擬音樂廳
聲音：自然、寬廣
Framing：古典、正式、「高級」
使用：想讓聲音有空間感但不突兀
```

### Chamber Reverb

```
歷史：真實的 echo chamber 房間
聲音：溫暖、中等密度
Framing：經典流行、60s-70s
使用：想要自然但有特色的空間
```

### Gated Reverb

```
歷史：1980s Phil Collins 鼓聲
聲音：爆發後突然切斷
Framing：80 年代、Power、懷舊
使用：想要戲劇性、年代感
```

### Shimmer Reverb

```
歷史：Brian Eno / Ambient 音樂
聲音：Pitch-shifted 的殘響尾巴
Framing：空靈、神聖、超自然
使用：想讓聲音「昇華」
```

## 實際應用：選 Reverb 的思路轉變

### 舊思路

```
這是人聲
→ 人聲通常在什麼空間？
→ 中型房間吧
→ 用 Room reverb
```

### 新思路

```
這首歌要傳達什麼？
→ 孤獨、夜晚、城市
→ 我要聽眾感受到什麼空間？
→ 不是「真實房間」，是「心理空間」
→ 短 reverb + 長 pre-delay + 暗 tone
→ 聲音清晰但有距離感
```

## Reverb 作為敘事工具

### 場景轉換

```
Verse：Dry（親密、直接）
    ↓
Chorus：Wet（開闊、史詩）
    ↓
Bridge：更 Wet + 特殊 reverb（夢境、轉折）
    ↓
Final Chorus：回到 Wet 但加東西（昇華）
```

Reverb 的變化 = 敘事的節奏

### 角色定位

```
主唱：少 reverb（前面、主角）
和聲：多 reverb（後面、配角）
合成器 Pad：很多 reverb（背景、氛圍）
```

Reverb 量 = 角色在舞台上的位置

## 連結：維根斯坦的 Picture Theory

（參見 [[music-framing-effect.md]]）

如果「框架是聲音的圖像」，那：

```
Reverb = 圖像的背景

它不是畫主體（聲音本身）
它是畫主體存在的世界
```

沒有背景的圖像是抽象的。
沒有 reverb 的聲音是「無處」的。

**Reverb 給聲音一個「所在」。**

但這個「所在」不必是真實空間，
它可以是情緒空間、文化空間、敘事空間。

## Pro-R 2 的設計哲學：佐證

FabFilter Pro-R 2 的設計其實暗示了這個觀點：

| 參數 | 傳統 reverb 沒有 | 意義 |
|-----|-----------------|------|
| Character | 不只是「空間類型」| 是「聲音個性」|
| Brightness | 不只是「材質模擬」| 是「情緒色彩」|
| Distance | 直接控制「遠近感」| 承認這是主觀參數 |

### Dan Worrall 的說明

關於 **Brightness** 參數：

> 「這個看似簡單的 Brightness 控制，以非常複雜而深刻的方式改變聲音⋯⋯**完全不像是在 EQ reverb**。」

關於 **Distance** 參數：

> 「Distance 參數⋯⋯控制 early 和 late 階段之間的平衡⋯⋯**但也控制前者如何過渡到後者**⋯⋯而且它**改變 reverb 的整體包絡**⋯⋯把聲源拉近，reverb 幾乎立即開始，帶有明顯的瞬態⋯⋯把它推遠，reverb 會更漸進地綻放。」

### 設計層面的認可

Pro-R 的參數命名選擇本身就是一種立場聲明：

```
傳統 reverb 參數          Pro-R 參數
────────────────         ────────────
Early reflection density  →  Character
High frequency decay      →  Brightness
ER/Tail balance          →  Distance
```

**它讓你調「感覺」，而不是調「物理模擬」。**

這是設計層面對「Reverb 作為框架」的認可。

---

## 結論

```
Reverb 的舊定義：聲音在哪裡
Reverb 的新定義：聲音屬於哪個世界
```

下次選 reverb 時，不要問：

> 「這個空間有多大？」

而是問：

> 「我要聽眾進入什麼樣的世界？」
