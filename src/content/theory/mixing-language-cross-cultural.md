(Mixing Vocabulary as Cultural Artifact)

描述聲音的詞彙不是中性的標籤,**是文化建構物**。每個語言提供不同的 vernacular vehicle,因此不同語言能輕易傳達的 qualia 是不同的。

**英文充滿髒話、身體、性、暴力隱喻;日文用擬聲擬態詞 + 食物 + 視覺;中文介於兩者之間。** 這不是巧合——是各自的 vernacular 文化資源決定的。

這是 [mixing-as-refined-intuition.md](/theory/mixing-as-refined-intuition/) 「Vernacular 作為知識載體」段落的延伸——把跨語言維度單獨拆出來討論。

## 核心觀察

### 英文 mixing 語彙的特徵

```
"fucking truly" "punchy" "fat" "phat" "thicc" "tits"  "bangin'"
"slamming" "knockin'" "hot" "wet" "gnarly" "filthy" "dirty"
"smacking" "biting" "in your face" "knock you in the chest"
```

充滿:
- **髒話**(fuck、shit、ass)
- **身體 / 性 / 暴力**(knock、slam、hot、wet、banging)
- **AAVE 黑人英語**(phat、fly、sick、dope、tight)
- **工人階級非正式 register**(gnarly、filthy)

### 日文 mixing 語彙的特徵

```
キンキン (kin-kin)        — 高頻刺耳
ボワッ (bowa)             — 低頻轟隆
シャリシャリ (shari-shari) — 過度高頻(像吃乾物)
モコモコ (moko-moko)       — 中頻悶住
ヌケがいい (nuke-ga-ii)    — 切過去 / cut-through 好
カラッとした              — 乾燥乾淨(像炸物)
煌びやか / きらびやか       — 璀璨閃耀(視覺)
迫力 (hakuryoku)          — 迫人氣勢
太い (futoi)              — 厚實
透明感 (toumeikan)        — 透明感
艶 (tsuya)                — 光澤
```

明顯特徵:
- **零髒話**——不是禮貌問題,是 register 從來沒徵召髒話進來
- **大量擬聲擬態詞(オノマトペ)**——日文獨有的密集語彙範疇,英文只有少數(boom、bang、splash),日文有 1000+
- **食物 / 觸覺隱喻**(カラッと = 像炸物、シャリシャリ = 像啃乾物)
- **視覺隱喻**(煌びやか、艶、透明感)
- **保留敬語邊界**——即使 casual register 也不破壞語域

### 中文 mixing 語彙的特徵

```
飽滿、乾淨、清晰、亮、暗、厚、薄、肥
爆、炸 (作為正面)
Q彈、嚼勁 (食物,質地)
顆粒感 (觸覺)
通透、清透 (視覺)
他媽的好聽 (作為強調詞,但不是描述詞)
```

特徵:
- **食物隱喻**(Q 彈、肥、嚼勁)——中文獨特的食物質地詞彙密度
- **視覺 + 物質隱喻**(通透、亮、厚)
- **單字化**——一個字承擔意義(亮、暗、厚)
- **髒話只當 intensifier,不當 descriptor**——「他媽的好聽」可以,但「fucking phat」這種**把髒話直接黏到描述詞上**的結構在中文不自然

## 三語對照表

| 英文 | 日文 | 中文 | 隱喻來源 |
|------|------|------|---------|
| punchy | パンチがある (loanword) / アタックが強い | 有力、夠力、衝 | 拳擊 → 身體衝擊 |
| fat / phat | 太い / ファット (loanword) | 肥、厚 | 身體質量 |
| dirty / filthy (positive) | 汚い (語感負面,不適用) | 髒(罕用作正面) | 道德污染 → 質地 |
| bangin' / slappin' | (難譯) | 炸、爆 | 暴力 → 衝擊 |
| fucking truly | 本当に / マジで | 真他媽的 | 髒話 → 強調 |
| sick / dope (positive) | ヤバい (positive sense) | 屌、爽、神 | 病態 / 危險 → 卓越 |
| gnarly | (無對應) | (無對應) | 衝浪文化 |
| **キンキンする** (glaring high) | — | 太亮、刺耳(較弱) | 擬聲 |
| **ボワッとする** (boomy low) | — | 悶、轟隆(較弱) | 擬聲 |
| **カラッとした** (dry/crisp) | — | 乾爽、乾淨(較弱) | 食物觸覺 |
| **煌びやか** (sparkling) | — | 通透、璀璨(略弱) | 視覺 |
| **Q 彈** | — | — | 食物彈性 |
| **嚼勁** | — | — | 食物嚼感 |

注意黑體那幾排——**三個語言都各有對方無法簡潔表達的範疇**。日文的擬聲擬態詞英文要寫一段;中文的「Q 彈」英日都要繞著說;英文的暴力性 / 性 metaphor 日中文都要消化過才能近似。

**沒有任何一個語言是「完整的」**——每個只是聲音空間的一個部分投影。

## 為什麼三個語言走向不同

### 英文的源頭: AAVE + 黑人音樂工業

**英文 mixing 詞彙不是「英美中產白人」的英文,是 AAVE(非裔美國黑人英語)+ 街頭文化的英文**。

關鍵歷史:
- Mixing 作為專業在 60s-70s 發展,主要場域是 R&B / soul / funk / 後來的 hip-hop
- 這些 genre 的 producer / engineer / artist 母語是 AAVE
- "phat"、"fly"、"dope"、"sick"、"fresh"、"tight" 全部是 AAVE 給流行語彙的捐獻
- 髒話的 normalisation 也是嘻哈文化的影響——hip-hop 把 "motherfucker" 變成可放歌詞、訪談、studio context 的詞

換句話說: **英文 mixing 語彙背後站著黑人美國文化**。Jaycen、Pensado 這些頂端 mixer 多半是黑人不是巧合——他們是這個語彙圈的母語使用者。**Mixing English ≠ Standard English**。

### 日文的源頭: 敬語 + オノマトペ + 美學傳統

日文沒走髒話路線,有三個結構原因:

1. **敬語體系太強**——日文的社會層級在語法上強制體現(です・ます vs 平敬 vs 尊敬語 vs 謙讓語)。在 studio 即使最 casual 的 register,也不會輕易破壞——因為 sempai-kohai 結構才是核心的「professional」訊號。Engineer 罵髒話會被視為失格而非親密。

2. **擬聲擬態語(オノマトペ)接管了 vernacular 的位子**——日文有 1000+ 擬音 / 擬態詞,英文只有少量。**這些詞已經承擔了「描述身體感受」的功能**,所以日文不需要徵召髒話來做這件事。「キンキンする」(高頻刺耳)的功能跟英文 "that fucking bites" 一樣——但載體不同。

3. **美學傳統的視覺 / 食物 / 季節比喻**——日文「煌びやか」「透明感」「カラッと」「艶」回到 1000 年前的 *物の哀れ* 美學語彙。聲音美學跟視覺、食物、季節美學共享同一個詞彙池。

### 中文的源頭: 食物 + 簡潔 + 工業歷史短

中文 mixing 語彙的特點:

1. **食物隱喻**(Q 彈、肥、嚼勁)——中文人對食物質地的描述比多數語言精細,這個資源外溢到聲音
2. **單字化**(亮、暗、厚)——中文一字一義,描述聲音直接挪用形容物質、空間、視覺的一字
3. **髒話是 intensifier 不是 descriptor**——「他媽的好聽」可以,但「fucking phat」這種**把髒話直接黏到描述詞**的結構在中文不自然。中文髒話結構偏 modifier/strengthener。
4. **流行音樂工業歷史較淺**(滾石、飛碟 80s 才崛起),mixing 詞彙還在發展。許多術語直接 loanword(「Compressor」「EQ」「mix」「掛 plugin」)

## Whorfian implication: 語言塑造你能感知的東西

這個觀察不是只是「翻譯問題」——是 **linguistic relativity**(語言相對論)的具體案例。

> 你的語言提供的詞彙範疇,**會反過來影響你優先注意到聲音的哪些屬性**。

具體例子:

- **日本 engineer 因為有「キンキン」這個詞**,對「高頻刺耳」這個範疇的辨識更早、更精細——他們聽到一點點 8 kHz 過量就會 trigger 這個詞,然後立刻想處理。英文 mixer 要描述同樣現象需要 "harsh", "brittle", "sibilant" 等多個詞合用,辨識精細度反而較低。
- **英文 mixer 因為有「punchy」這個身體衝擊詞**,對 transient 的身體感受更精細——他們會在 attack envelope 上花更多時間,因為這個範疇有專屬詞。日本 engineer 用「アタックの強い」(loanword)描述同樣現象,語感較弱,可能花較少時間。
- **中文 mixer 用「Q 彈」描述 bass 質感**,這個詞綁定的是 chewy / bouncy 的口感——所以中文 mixer 在做 bass 時可能無意識追求 "chewy" 質地,這在英文 / 日文裡不是常見追求。

這不是迷信。**詞彙提供 attentional spotlight**——你有名字的東西,你看得見;你沒名字的東西,你聽得見但不會處理。

## 對 mixing 教育的 implication

這也解釋了為什麼**翻譯版混音教材常常「失味」**:

| 原文 | 翻譯後常變成 |
|------|--------------|
| "Make this kick fucking knock" | 「讓這個 kick 更有力」 |
| "Pull the mid out of this guitar, it's getting nasty" | 「降低吉他的中頻,它太刺耳」 |
| "We want this vocal to slap you in the face" | 「我們希望這個人聲很有衝擊力」 |

**身體 / 暴力 / 性的 register 全部被洗掉**,變成「中性 step 2 register」。學生讀到的是規則描述,不是 qualia 描述。所以**讀英文翻譯教材的學生比讀英文原版的學生,少接觸到 step 3 的 vernacular 訊息**——這是中文 / 日文混音學習者的隱性劣勢。

反過來: **日文混音教材對英文母語使用者也有一樣的損失**——「キンキン」「カラッ」這些日文擬音語的精度跟身體連結性,翻成英文後變得 schematic。

## 對你的 implication

你做 J-pop / Mandopop 雙語原創——所以你需要**三套詞彙在腦中共存**:

| 語彙池 | 用途 |
|--------|------|
| **英文 mixing 詞彙** | 跟英美 reference 對照、讀英文教材、跟 plugin manuals 對接 |
| **日文 mixing 詞彙** | 做 J-pop 時的 vernacular 接收器——讓キンキン / ボワッ / カラッ 進入身體訊號詞彙 |
| **中文 mixing 詞彙** | 做中文歌時的 vernacular 接收器——Q 彈、通透、肥 是你母語級的 qualia 載體 |

具體可以做的:

1. **寫「身體筆記」時容許混合語言**——不要只用英文記 "this needs more punch",可以記:
   - 「這個 kick 不夠 Q 彈」
   - 「vocal キンキン了」
   - 「mix 太肥」
   - 「snare カラッと不夠」

   讓三種 vernacular 同時運作,你的 S1 接收訊號的 bandwidth 才完整。

2. **建立平行字典**:同一個 qualia 在三語怎麼說,寫下來。例如:
   - 英: "this kick needs to slap"
   - 日:「キックにもう少し迫力が欲しい」
   - 中:「kick 要再有力一點 / 衝一點」
   
   三個版本身體 trigger 不同強度——三個都 internalize,你的 polyphonic vernacular 才建起來。

3. **刻意切換 register 校準 mix 方向**——如果你做 J-pop 卻只用英文 vernacular 思考,**你會無意識把 J-pop mix 推向 American pop 的 aesthetic**(因為英文的「punchy / fat / fucking knock」這些詞身體記憶 = American pop 的 transient + sub heavy 模板)。**刻意切到日文「キンキンしないように、太すぎず、もっと透明感を」思考,會把 mix 拉回 J-pop register**。

## 討論區

### 問題: 韓文 mixing 語彙呢?
不在我熟悉範圍內,但 K-pop 這 15 年的工業化過程值得單獨研究。猜測: 韓文有大量擬聲詞傳統(類似日文),但 K-pop 全球化導致大量 loanword 從英文進來。可能正在形成 hybrid 語彙(英文身體詞 + 韓文擬聲詞)。

### 問題: 我可以「translate vernacular」嗎?
不能完全。但你可以**學會同一個 qualia 在三個語言的不同入口**。例: 英文的 "this needs to slap" / 日文的 "もう少しパンチが欲しい" / 中文的 "再有力一點" 都指向同個 qualia,但身體 trigger 不同強度。三個都 internalize,你的 polyphonic vernacular 就建起來。

### 問題: 我寫日文歌詞,我用英文罵聲音的習慣會干擾嗎?
有可能。如果你習慣英文 vernacular 描述聲音,你的 S1 在 J-pop session 上可能 default 到英文 reference,結果 mix 出來太 American——對 J-pop 而言失去某些日式微妙(透明感、繊細さ、艶)。**刻意切到日文 vernacular 思考是 register 校準工具**。

### 問題: 為什麼台灣 / 香港 mixing 學生常感覺英文教材「不夠到位」?
不一定是英文不好,**可能是 vernacular 沒接到**。書面英文 mixing 教材常常已經是 step 2 register(因為要可讀、可規範化),vernacular 已經被脫水。要看 vernacular 原樣,得看 podcast、訪談影片、Workshop 這種 face-to-face 場合的非書面英文。
