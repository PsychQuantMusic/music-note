EQ 的傳統定義是「頻率校正」。但這個定義太淺了。

**EQ 不是修正頻率，是選擇聲音如何「說話」。**

---

## 兩種思考方式

### 傳統：EQ = 頻率校正

```
問題：這個聲音有什麼問題？
答案：太糊 → 切 300 Hz
      太刺 → 切 3 kHz
      太悶 → 加 10 kHz
```

這是「修復」的思路——聲音有問題，我來修。

### 新觀點：EQ = Framing

```
問題：我要這個聲音「說什麼」？
答案：要溫暖 → 強調 200 Hz 的「身體」
      要親密 → 保留 proximity effect 的近距離感
      要現代 → 強調「空氣感」和「清晰度」
      要復古 → 滾掉高頻，讓它「舊」
```

這是「選擇」的思路——我在決定這個聲音的性格。

---

## 證據：我們用來描述 EQ 的詞彙

看看混音師怎麼談論頻率：

| 頻率範圍 | 技術描述 | 實際使用的詞彙 |
|----------|----------|----------------|
| 60-250 Hz | 低頻 | **溫暖**、**厚度**、**punch** |
| 250-500 Hz | 中低頻 | **泥濘**、**箱子聲**、**body** |
| 2-6 kHz | 中高頻 | **清晰度**、**presence**、**刺耳** |
| 10k+ Hz | 高頻 | **空氣感**、**sparkle**、**光澤** |

**這些詞彙沒有一個是物理術語。**

它們全都是：
- 隱喻（溫暖、冷、暗、亮）
- 情緒（刺耳、甜美）
- 空間概念（空氣、body）

**當我們調 EQ 時，我們不是在「調頻率」，我們是在「選擇聲音的性格」。**

## 外部資料佐證

### Dan Worrall（FabFilter Pro-Q 4 Introduction）

關於 Pro-Q 4 的 Character 模式：

> 「**Pro-Q 4 提供兩種新的 Character 模式**：'Subtle' 增加一整組諧波⋯⋯而 'Warm' 給你一個一致的、悅耳的二次諧波⋯⋯**這些是 oversampled 的，而且可以與任何處理模式搭配**，所以你確實可以有**有色彩的線性相位 EQ**，如果你想要的話。」

> 「當然，你還是需要有意識地啟用這些設定⋯⋯除非你選擇你喜歡的風味，然後把它存成預設⋯⋯**現在你預設就有一點色彩，就像類比 console 一樣**。」

注意：這個 EQ 插件明確提供「Character」（性格）模式，而不是「Correction」（校正）模式。

**設計者認可 EQ 不只是修正工具，是塑造性格的工具。**

### Dan Worrall（The Hardware Sound）

關於「類比聲音」的本質：

> 「**當一個錄音變成文化標誌，它的缺陷就不再是缺陷了。它們變成特色。**」

> 「John Bonham 的鼓在 50 Hz 需要更多下潛。⋯⋯Stevie Wonder 的 Sir Duke 裡的銅管太薄太乾。⋯⋯抱歉，**事實不在乎你的感受**。」

> 「所以，達成你說的那種 hardware/analog 聲音的方法是：寫一首絕對的神曲，錄下來⋯⋯確保它大獲成功，最好能定義一整個文化時代⋯⋯然後幾十年後⋯⋯告訴他們你用了硬體。」

這段話的意義：

```
「好聽的 EQ」不是客觀存在的
它是文化建構的
「經典聲音」= 被集體記憶神聖化的特定 EQ 特徵
```

**我們追求的「類比溫暖」，其實是在追求一種文化符號。**

### Dan Worrall（關於 EQ 的核心地位）

> 「**到頭來，訓練你的耳朵設定好的平衡和善用 EQ，這幾乎就是全部了**。不要讓 GAS（器材焦慮症）告訴你其他的。」

這暗示 EQ 不只是眾多效果之一，它是混音的核心修辭工具。

### Pro-Q 4 的設計哲學

Pro-Q 4 的 Spectral Dynamics 功能說明：

> 「如果**懶惰或不適當地使用**，這種處理絕對能夠**抑制你音軌中所有的性格和生命力，把讓它成為它自己的諧波都壓下去**。」

> 「⋯⋯你也可以用擴展（expansion）來**誇大自然特徵**⋯⋯聽聽那些一直藏在 Rhodes 裡的瘋狂母音聲音⋯⋯」

這段話的關鍵詞：

- **character**（性格）
- **life**（生命力）
- **natural characteristics**（自然特徵）

**EQ 可以抑制或誇大聲音的「性格」。這不是頻率校正，這是身份塑造。**

## EQ 參數的 Framing 意義

### Low-End（低頻）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 厚重低頻 | 更多 sub | 力量、沉穩、權威 |
| 乾淨低頻 | 只保留 fundamental | 現代、精準、控制 |
| 滾掉低頻 | 缺乏低頻 | 距離感、「小」、電話感 |

### Midrange（中頻）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 飽滿中頻 | 2-4 kHz presence | 親密、「在你面前」 |
| 凹陷中頻 | scooped mids | 「金屬」、侵略性、距離 |
| 鼻音中頻 | 800 Hz honk | Lo-fi、復古廣播 |

### High-End（高頻）

| 設定 | 物理意義 | Framing 意義 |
|-----|---------|-------------|
| 明亮 | 多高頻 | 現代、清晰、年輕 |
| 暗 | 少高頻 | 復古、溫暖、夜晚 |
| 空氣感 | 10k+ boost | 「昂貴」、「專業」、開放 |

## 頻率區域的文化符號

不同頻率攜帶不同的文化意義：

### 「溫暖」（100-300 Hz）

```
歷史：類比設備的自然滾降、磁帶的飽和
聲音：豐滿、圓潤、有「身體」
Framing：復古、人性、「真實」
使用：想讓聲音聽起來「經典」
```

### 「Presence」（2-4 kHz）

```
歷史：人耳最敏感區域、語言清晰度的關鍵
聲音：靠近、清晰、「在你面前說話」
Framing：親密、直接、真誠
使用：讓人聲「穿透」混音
```

### 「Air」（10k+ Hz）

```
歷史：高品質錄音和昂貴麥克風的特徵
聲音：開放、細節、「呼吸感」
Framing：現代、專業、「高端」
使用：讓混音聽起來「昂貴」
```

### 「Telephone」（300-3k Hz）

```
歷史：電話的頻寬限制
聲音：中頻主導、無高低頻
Framing：距離、回憶、另一個時空
使用：故意的 lo-fi 效果、回憶場景
```

## EQ 作為「選擇措辭」

### 語言學類比

| 語言 | EQ |
|------|-----|
| 選詞（正式 vs 口語）| 選擇頻率特徵（亮 vs 暗）|
| 語氣（溫和 vs 尖銳）| 中高頻平衡（柔和 vs harsh）|
| 強調（重音）| Presence boost（讓它「說話」）|
| 省略（不說的）| High-pass, notch cut（不讓它說）|

### 例子：同一段人聲，不同 EQ = 不同「說法」

| EQ 設定 | 聲音「說」的話 |
|---------|---------------|
| 溫暖、proximity effect | 「我在你耳邊說悄悄話」 |
| Presence boost、明亮 | 「聽我說！這很重要！」 |
| Telephone EQ | 「這是一段回憶⋯⋯」 |
| 暗、滾高頻 | 「夜深了，我很疲憊」 |
| Harsh、過多 3k | 「我很焦躁、緊張」 |

**你在選擇聲音「用什麼語氣說話」。**

## 實際應用：選 EQ 的思路轉變

### 舊思路

```
這個人聲有點悶
→ 問題：缺乏清晰度
→ 解法：加 3 kHz
→ 完成
```

### 新思路

```
這首歌要傳達什麼？
→ 深夜的孤獨、疲憊的告白
→ 我要人聲聽起來怎樣？
→ 不是「清晰」，是「疲憊」
→ 不加 presence，反而滾一點高頻
→ 讓它「暗」，配合歌曲的情緒
```

## EQ 作為敘事工具

### 情緒變化

```
Verse：暗、親密（悄悄話）
    ↓
Pre-chorus：漸亮（能量上升）
    ↓
Chorus：明亮、presence（爆發）
    ↓
Bridge：更暗、lo-fi（反思）
    ↓
Final Chorus：最亮 + air（昇華）
```

**EQ automation = 敘事的情緒弧線**

### 角色定位

```
主唱：最飽滿的頻率分布（主角）
和聲：切掉一些 presence（配角、不搶戲）
背景樂器：narrower 頻寬（場景、不是人物）
```

## 連結：維根斯坦的語言遊戲

（參見 [[music-framing-effect.md]]）

如果「音樂是語言遊戲」，那 EQ 就是：

```
EQ = 選擇「詞彙」和「語氣」

你不是在修正頻率
你是在選擇聲音「說話的方式」
```

### 不同「遊戲」的 EQ 規則

| 曲風 | EQ「遊戲規則」 |
|------|----------------|
| 現代流行 | 亮、清晰、大量 air |
| Lo-fi Hip-hop | 暗、滾高頻、vinyl crackle |
| 80s Synth-pop | scooped mids、誇張高低頻 |
| Folk/Acoustic | 自然、minimal EQ、「真實」|
| Heavy Metal | 極端 scooped mids、「牆壁」感 |

**違反這些規則不是「錯」，是「說了不合時宜的話」。**

---

## EQ vs Reverb：兩種 Framing

| Reverb | EQ |
|--------|-----|
| 建構「空間」| 建構「性格」|
| 聲音「在哪裡」| 聲音「是什麼」|
| 物理位置的隱喻 | 情緒/身份的隱喻 |
| 圖像的「背景」| 圖像的「筆觸」|

兩者都是 Framing 工具，但操作不同維度：

```
Reverb：這個聲音存在於什麼世界？
EQ：這個聲音有什麼性格？
```

---

## 結論

```
EQ 的舊定義：修正頻率問題
EQ 的新定義：選擇聲音的性格和語氣
```

下次調 EQ 時，不要只問：

> 「這裡有什麼問題要修？」

而是問：

> 「我要這個聲音說什麼？用什麼語氣說？」
